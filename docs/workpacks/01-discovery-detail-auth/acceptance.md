# Acceptance Checklist

## Happy Path
- [ ] HOME 목록이 보인다
- [ ] 검색어 입력 시 제목 기준으로 목록이 갱신된다
- [ ] 정렬 변경 시 목록 순서가 갱신된다
- [ ] 카드 선택 시 RECIPE_DETAIL로 이동한다
- [ ] 비로그인 상태에서 보호 액션 시 로그인 모달이 열린다
- [ ] 로그인 후 원래 레시피 상세 화면으로 복귀한다

## State / Policy
- [ ] 조회는 비로그인 허용이다
- [ ] 보호 액션은 로그인 전 실행되지 않는다
- [ ] 요리모드/쓰기 액션은 이번 슬라이스 밖으로 남겨둔다

## Error / Permission
- [ ] HOME 에러 상태가 있다
- [ ] HOME 빈 상태가 있다
- [ ] 상세 에러 상태가 있다
- [ ] Supabase 설정 누락 시 API가 실패 응답을 준다

## Manual QA
1. HOME에서 검색과 정렬을 바꿔본다.
2. 상세에서 보호 액션을 눌러 로그인 게이트를 확인한다.
3. 소셜 로그인 후 같은 레시피로 복귀하는지 확인한다.
