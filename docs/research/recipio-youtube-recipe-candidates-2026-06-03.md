# Recipio YouTube Recipe Extraction Candidates

조사일: 2026-06-03 KST

## 목적

레시피오에 등록된 YouTube 기반 레시피 중, 현재 저장소에서 이미 사용된 YouTube ID가 아닌 새 검증 후보를 조건별로 정리한다. 후보는 공개 Recipio 상세 페이지의 `Recipe` JSON-LD와 Chrome에서 직접 확인한 YouTube 설명란/댓글 영역을 기준으로 분류했다.

## 제외 기준

- 저장소 전체에서 `rg`로 확인된 기존 YouTube ID는 제외했다.
- 아래 표의 모든 YouTube ID는 현재 저장소에서 검색되지 않았다.
- Recipio `/api/` 경로는 `robots.txt`에서 제외되어 있어 사용하지 않았다. 공개 RSS, 공개 상세 HTML, 공개 YouTube 페이지 확인만 사용했다.

## 활용 메모

- 조건 1은 설명란 파싱 정확도 회귀 테스트에 적합하다.
- 조건 2는 댓글 수집, 고정댓글 우선순위, 작성자 댓글 판별 테스트에 적합하다.
- 조건 3은 설명/댓글 텍스트 없이 자막 또는 화면 OCR 기반 추출이 얼마나 구조화되는지 확인하는 테스트에 적합하다.
- 조건 3의 `J5Rmux3ttaY`는 시청자 댓글이 많으므로, 댓글 텍스트가 많아도 작성자 레시피 본문이 없을 때 자막/OCR 후보로 분류되는지 확인하기 좋다.
