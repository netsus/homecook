## 잠금 요약

- 하단 CTA는 `[적용]` / `[초기화]`로 고정한다.
- `count_only`, `ingredients`, `[레시피 N개 보기]` 같은 비공식 계약은 사용하지 않는다.
- `전체`는 UI 전용 sentinel이며, 전체 조회는 `category` 미전달로 처리한다.
- 카테고리 탭 전환 시 검색어는 유지되며 `q + category` AND 조회를 수행한다.
- 선택 UI는 **체크리스트 의미**를 유지하고, 시각 표현만 pill/chip 형태를 사용한다.

## 컴포넌트 명세

### BottomSheet 컨테이너

- 배경: `--panel`
- 최대 높이: `85vh`
- 상단 radius: `20px`
- 오버레이 탭 / 닫기 버튼 / 드래그 다운으로 닫을 수 있다
- 닫기 시 draft 선택은 버리고 HOME의 applied 상태를 유지한다

### 검색 입력

- placeholder: `재료명으로 검색`
- 디바운스 300ms
- active category가 `전체`가 아니면 `q + category` AND 조회
- 검색어 초기화 시 현재 active category 기준으로 다시 조회

### 카테고리 탭

- 탭 목록: `전체 / 채소 / 육류 / 해산물 / 양념 / 유제품 / 곡류 / 기타`
- `전체`는 UI sentinel이다
- `전체` 활성 시 서버에는 `category`를 보내지 않는다
- 다른 탭 활성 시 `category=<selected>`를 사용한다
- 탭 전환 시 검색어는 유지한다

### 재료 체크리스트

- 의미상 체크리스트, 시각상 pill/chip button 사용 가능
- 각 항목은 `min-h: 44px`를 보장한다
- 미선택: `--surface` 배경, `--line` border
- 선택: `--olive` 배경, 체크 아이콘 포함, 흰색 텍스트
- 내부 아이콘은 `14px`, 텍스트와 `gap-1`

### 하단 버튼 영역

- 안내 문구
  - 0개 선택: `재료를 선택하면 레시피를 필터링해요`
  - N개 선택: `N개 선택됨`
- `[초기화]`
  - draft 선택값만 초기화
  - 0개 선택 시 비활성
- `[적용]`
  - label 고정, preview count 없음
  - draft 선택값을 HOME applied 상태에 반영
  - 적용 직후 HOME이 `ingredient_ids` same-session mirror를 URL에 동기화

## 상태 정의

### Loading

- 모달 진입 시 재료 목록 로딩
- 검색 / 카테고리 전환 중에는 체크리스트 영역만 skeleton 표시

### Empty

- `검색 결과가 없어요`
- `다른 재료명으로 검색해보세요`

### Error

- `재료 목록을 불러오지 못했어요`
- `[다시 시도]`로 현재 `q + category` 조합 재호출

## 인터랙션 노트

| 액션 | 결과 |
| --- | --- |
| 모달 열기 | draft 선택값을 HOME applied 상태에서 복사하고, 현재 `q / category` 기준으로 `GET /ingredients` 조회 |
| 검색어 입력 | 디바운스 300ms 후 `GET /ingredients?q=...` 조회 |
| 카테고리 탭 전환 | 기존 검색어를 유지하고 `q + category` AND 조회 |
| 재료 선택/해제 | draft 선택값만 토글, 레시피 preview API 호출 없음 |
| 초기화 | draft 선택값만 비우고 모달 내부 상태 갱신 |
| 적용 | HOME applied 상태 반영 → 모달 닫기 → HOME 목록 재조회 |
| 닫기(취소) | draft 선택값 폐기, HOME applied 상태 유지 |
| 재시도 | 현재 `q / category` 조건으로 `GET /ingredients` 재호출 |

## 디자인 결정 사항

- 공식 화면정의서의 `[적용]` / `[초기화]`를 우선한다.
- `previewCount` / `count_only` / `ingredients` query는 사용하지 않는다.
- `전체` 탭은 필수 UX이지만 서버 계약값이 아니므로 UI sentinel로만 둔다.
- 체크리스트 의미를 우선하고, 공간 효율 때문에 시각 스타일만 pill/chip으로 사용한다.
- 검색 중 카테고리를 바꾸면 검색어는 유지한다.
- `PANTRY` 경유 진입은 이번 슬라이스 범위에 포함하지 않는다.
